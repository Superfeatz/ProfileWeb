  • Machine Learning Image Classification

    Proyek klasifikasi sampah menggunakan Support Vector Machines (SVM) dengan kernel Linear, Polynomial, RBF, dan Sigmoid. Proyek ini mencakup pra-pemrosesan data (resize gambar, hapus file rusak), pelatihan model menggunakan scikit-learn, dan evaluasi performa dengan metrik akurasi, laporan klasifikasi, dan matriks kebingungan. Dataset diambil dari Kaggle, dan kode tersedia dalam bentuk Jupyter Notebook (svm.ipynb) dan skrip Python (svm.py). Model yang telah dilatih disimpan sebagai file .p, dan fitur gambar diekstrak menggunakan img2vec untuk prediksi.


    Lihat di GitHub
